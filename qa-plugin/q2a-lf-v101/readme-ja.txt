/*******************************************************************/
言語フィルター（置換）プラグイン for question2answer
/*******************************************************************/

/*-----------------------------------------------------------------*/
1. 概要
/*-----------------------------------------------------------------*/
このパッケージはquestion2answer用のプラグインです。

question2answer: http://www.question2answer.org/

/*-----------------------------------------------------------------*/
2. このプラグインの機能
/*-----------------------------------------------------------------*/
1.Q2Aの言語質問に拡張フィールドを追加できます。

/*-----------------------------------------------------------------*/
3. バージョンの互換性
/*-----------------------------------------------------------------*/
question2answer V1.5以上

/*-----------------------------------------------------------------*/
4. インストール／設定
/*-----------------------------------------------------------------*/
1.アーカイブファイルを解凍します。
2.qa-pluginフォルダの下にlang-filterフォルダをアップします。
3.管理者権限でログインします。
4.「管理」->「プラグイン」メニューを選択します。
5.オプションを設定してから保存します。

/*-----------------------------------------------------------------*/
5. アンインストール
/*-----------------------------------------------------------------*/
1.管理者権限でログインします。
2.「管理」->「プラグイン」メニューを選択します。
3.「リセット」ボタンをクリック
4.qa-pluginフォルダの下のlang-filter フォルダを削除します。

/*-----------------------------------------------------------------*/
6. オプション説明
/*-----------------------------------------------------------------*/
[プラグインを有効にする]
OFF: プラグインが無効になります。
ON:  プラグインが有効になります。

[置換する語句:]
置換する語句のペアを入力します。
変換前==変換後（||で連結。改行は||に自動変換されます）

入力例(ルール):
手動設定の例: 質問==トピック||回答==返信
プログラム設定の例: @EVAL return '質問==トピック||回答==返信';

/*-----------------------------------------------------------------*/
7. プラグインを強制的に無効にする方法
/*-----------------------------------------------------------------*/
間違った置換語句を入力した場合等でQ2Aが誤動作した場合は、phpMyAdminを
使って、プラグインの有効フラグを落としてください。

テーブル: qa_options
title : lang_filter_enable
content: 1 -> 0
